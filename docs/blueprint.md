# **App Name**: StreamerU Hub

## Core Features:

- Homepage Hub: Display a dynamic homepage with key information. Focus on clear calls to action like 'Enroll Now' and 'Become a Professor'.
- Interactive Course Listings: Implement a course catalog page with filterable options.
- Responsive Design: Ensure all pages are responsive across desktop, tablet, and mobile devices.
- AI-Powered Professor Bio Tool: Implement a 'Professor Bio Generator' tool. New professors can input keywords about their expertise and generate a compelling bio.

## Style Guidelines:

- Primary color: #8B0000 (Dark Red) for a bold and academic feel. This hue evokes a sense of ambition and focus that’s perfect for the university atmosphere.
- Background color: #F5F5F5 (Light Gray) for a clean, modern look.
- Accent color: #D4A27A (Beige) to complement the red.
- Use a clean and readable font for body text, like Open Sans.
- Use a bold and modern font for headings, such as Montserrat.
- Incorporate relevant icons (graduation cap, book, etc.).
- Maintain a clean, grid-based layout with clear sections.
